## Descripción del Proyecto 
* Fuente de datos
* El set de datos lo obtuvimos del censo de población y vivienda del INEGI realizado en 2020. El proposito de nuestra recolección de datos es ver las diferencias de acceso a la salud y servicios básicos disponibles por edad, ocupación y zona geográfica. Los datos del censo se actualizan cada 10 años. Obtuvimos los datos de https://www.inegi.org.mx/app/scitel/Default?ev=9.
